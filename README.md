📈 Real-Time Stock Market Dashboard

A web-based dashboard to track and visualize stock market data in real time.

🚀 Features

- 📊 Interactive stock price charts
- 📈 Moving average analysis
- 🌍 Supports global stocks (US & India)
- 🔍 User input for stock symbols
- ⚡ Real-time data using yfinance

🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- yfinance

📦 Installation

1. Clone the repository:
git clone https://github.com/thenavikamal67/stock-dashboard
2. Navigate to project folder:
cd stock-dashboard
3. Install dependencies:
pip install -r requirements.txt
4. Run the app:
streamlit run app.py

📊 Example Stocks

- AAPL (Apple)
- TSLA (Tesla)
- MSFT (Microsoft)
- RELIANCE.NS (India)
